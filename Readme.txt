Technologies Used

1. Spring Boot (Java-based framework)
2. MongoDB (NoSQL database)
3. Spring Data MongoDB (MongoDB integration with Spring Boot)
4. JPA for database queries

Setting up the project

Clone the project from the GitHub or extract from the zip folder provided.
Make sure MongoDB is installed and running.

Run the Application

Download the extension - Extension Pack for Java and Spring Boot Extension Pack
After installing, click on the spring boot icon and then search for the app name demo and run it.

or

you can install the maven from the internet and setup the environment variables of the bin folder.
Then, type in cmd of the project mvn clean install and then to run mvn spring-boot:run
